## Project Overview

DianiVet is a modern, responsive veterinary clinic website built with **Next.js**.  
The project is designed to showcase the clinic’s services, team, and facilities while providing an easy way for users to **book appointments** and **contact the clinic**.  

This project emphasizes user experience, clean design, and responsive layouts to ensure accessibility on both desktop and mobile devices.

**Live Site:** [https://vetshop.vercel.app](https://vetshop.vercel.app)

---

## Features

- **Homepage** with hero sections, features, and galleries
- **About Page** introducing the clinic and its team
- **Services Page** detailing veterinary services offered
- **Contact Page** with appointment booking form
- **Responsive design** for mobile and desktop devices
- Modern UI with interactive components
- Smooth client-side navigation using Next.js

---

## Tech Stack

- **Framework:** Next.js 13 (App Router)
- **Languages:** JavaScript, JSX, CSS
- **Styling:** Tailwind CSS
- **Hosting/Deployment:** Vercel
- **Version Control:** Git & GitHub
